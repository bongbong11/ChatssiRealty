# 챗씨부동산 — 디자인 토큰 (CSS 분리용)

> 목업(home-closet-mockup.html)에서 추출한 값. 빌드시 `style.css`로 그대로 옮겨 쓰면 됨.
> 컨셉: 탭1 = 부동산 증서(Deed) 문서 느낌 / 탭2 = 귀여운 스크랩북·인벤토리 느낌
> 탭 전환 시 배경/폰트가 같이 바뀌면서 "공문서 ↔ 스크랩북" 분위기 전환이 시그니처 요소

## 폰트
```css
@import url('https://fonts.googleapis.com/css2?family=Fraunces:opsz,wght@9..144,400;9..144,600;9..144,700&family=Baloo+2:wght@500;700&family=Nunito:wght@400;600;700;800&display=swap');
```
- `Fraunces` — 탭1(부동산) 헤더/타이틀용 세리프, 문서 느낌
- `Baloo 2` — 탭2(소지품) 헤더용 둥근 폰트, 귀여운 느낌
- `Nunito` — 공통 본문/UI 텍스트

## 공통 컴포넌트 스타일 요약
- **앱 컨테이너**: max-width 420px, border-radius 28px, 짙은 배경 위에 떠 있는 카드형
- **탭 버튼**: pill 형태, active 시 진한 배경(doc-ink / cute-text)으로 반전
- **Deed 카드**: 우상단에 회전된 "DEED" 도장 워터마크(opacity .5), 점선 구분선 그리드
- **story-box**: 점선 테두리 박스, 라벨(골드) + 본문(작은 세리프 계열 텍스트)
- **토글 스위치**: 42x23px pill, on 상태에서 doc-gold로 채워짐
- **아이템 슬롯**: 1:1 정사각형, locked는 빗줄무늬 패턴 배경 + 자물쇠 이모지 + 해금가 배지
- **모달 카드**: 300px 폭, 상단 브랜드명(Baloo 2) → 가격(레드, bold) → TMI(베이지 배경 박스) → 핀버튼 → 닫기버튼
- **food-list (팬트리/냉장고)**: 카드 없이 줄 단위 리스트, 점선 구분, 특별 아이템만 해금 배지
- **subview(풀스크린 전환)**: 뒤로가기 버튼 + 제목 + 좌우 토글, 메인 그리드/포인트바는 완전히 숨겨짐

## 시그니처 인터랙션
- 탭 전환 시 `.app` 클래스에 `.cute` 토글 → 배경색/폰트 패밀리가 동시에 전환되며 부드러운 트랜지션(.45s)
- 잠금 아이템 클릭 무반응, 해금된 아이템만 클릭 가능 → 모달
- 핀 토글은 모달 내부와 그리드 양쪽에 동시 반영
